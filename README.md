# prod-destroyer

A tool for safely destroying production resources.

## Overview

prod-destroyer helps you manage and tear down production infrastructure in a controlled, auditable way.

## Features

- Controlled resource teardown
- Audit logging of all destructive actions
- Safety checks before any production changes
- Dry-run mode to preview what will be destroyed

## Usage

```bash
prod-destroyer --target <resource> --dry-run
prod-destroyer --target <resource> --confirm
```
